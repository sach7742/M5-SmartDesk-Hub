# 🤖 M5-SmartDesk Hub

A professional multi-function desk assistant for M5Stack robots. This project demonstrates modular state-machine logic, UI design, and hardware integration using C++.

## ✨ Features
- **Smart Clock**: A real-time analog clock display using trigonometric coordinate mapping.
- **M5-MemoRythm**: An integrated memory and rhythm game with synchronized audio-visual feedback.
- **Modular UI**: A clean navigation system using physical hardware buttons (A, B, and C).

## 🛠️ Technical Skills Demonstrated
- **Language**: C++ / Arduino
- **Architecture**: State-Machine Logic (for app switching)
- **Graphics**: Real-time rendering and coordinate geometry (Sin/Cos)
- **Version Control**: Managed via Git/GitHub

## 🚀 How to Run
1. Install the [M5Unified](https://github.com/m5stack/M5Unified) library.
2. Open `M5-SmartDesk-Hub.ino` in your IDE.
3. Upload to your M5Stack device.
